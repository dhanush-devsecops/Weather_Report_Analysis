[ PROJECT OVERVIEW ]
This project involves the end-to-end processing and analysis of daily weather 
records from Telangana. It bridges raw data engineering with predictive 
modeling to provide actionable weather intelligence.

Key highlights:
- Analysis of 35+ daily records across various Telangana districts.
- Data Preprocessing: Handling missing values via Mean Imputation.
- AI Prediction: Machine Learning model to forecast rain probability.
- Visualization: Interactive trend analysis and feature correlation.

[ TECH STACK ]
- Programming: Python
- Framework: Streamlit (Web UI)
- Libraries: Pandas, NumPy, Scikit-learn, Plotly, Seaborn
- Environment: Linux/Ubuntu (as seen in demo)

[ FILE STRUCTURE ]
- app.py            : Main Streamlit dashboard application
- data_cleaning.py   : Logic for cleaning and mean imputation
- analysis_viz.py    : Statistical plotting and visualization scripts
- requirements.txt   : List of project dependencies

[ HOW TO RUN ]
1. Open your terminal or command prompt.
2. Install the necessary dependencies:
   pip install -r requirements.txt

3. Launch the interactive dashboard:
   streamlit run app.py

[ KEY FEATURES IN DASHBOARD ]
1. Exploration Settings: Filter by District (e.g., Adilabad, Hyderabad).
2. AI Rain Prediction: Input Temp/Humidity/Wind Speed to get a forecast.
3. Feature Correlation: Heatmap showing relationships between weather variables.
4. Data Export: Download filtered results as a CSV file.
